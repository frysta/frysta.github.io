Commands

A list of commands and descriptions

Economy

In-bot currency system
  • =balance/ =bal [tagged user] - Check how much money you / the tagged user has
  • =beg - Beg for coins
  • =bet (amount of coins) - Bet coins
  • =buy (item) - Purchase an item from the shop
  • =daily - redeem a daily reward of 2000 coins
  • =inventory [tagged user] - Check an inventory
  • =leaderboard - See how rich you are in your server
  • =shop [item] - Check the shops for items
  • =work - Work for coins

Fun

Miscellaneous commands
  • =8ball (question) - Ask a yes/No question
  • =discriminator [discriminator] - Find other people by discriminator
  • =hug (tagged user) - Hug a user
  • =joke - hear a joke from reddit
  • =meme - Get memes from meme subreddits
  • =mime (tagged user) (message) - Mime a user with webhooks
  • =mimembed (tagged user) (colour of embed) (title of embed) (text in embed)
  • =raffle (number) - Pick a random number
  • =roast [tagged user] - Roast someone
  • =say (message you want Frysta to say) - Frysta says something
  • =showerthoughts - Get posts from r/showerThoughts
  • =tifu - Get posts from r/TIFU

Image

Image manipulation and search
  • =always (text) - Generate an astronaut meme (always has been) with your custom text
  • =anna [tagged user] - You become Anna
  • =clyde (text) - Make Clyde say something
  • =colour (colour hex or name) - See a colour
  • =detect (image URL) - detect the majority colour of an image
  • =drake [tagged user] - Generate a drake meme with your custom text
  • =elsa [tagged user] - You become Elsa
  • =image (search query) - Perform an image search
  • =quote (tagged user) (text) - Generate an image where a user says something
  • =stonks (text) - Generate a stonks meme with your custom text

Info

Obtain general information about Discord
  • =commands - Display a list of commands
  • =credits - See who contributed to make Frysta
  • =define [word] - Get the Urban dictionary defenition for a word
  • =emoji (emoji) - Get the emoji as a .png
  • =emojis - Get a list of all of the custom emojis in the server
  • =google (search query) - look something up
  • =help - Get links
  • =id (tagged user/channel/role) - Get ID
  • =info (user/server/frysta) - Get information about a user or the server or Frysta
  • =invite [bot ID] - Sends an invite link with the bot ID
  • =languages - Get a list of all language codes
  • =music - List of all music commands
  • =pfp [tagged user] - Get a link to a profile picture
  • =settings - View the settings for your server
  • =test - Check the ping of the bot

Moderation

Manage server responsibilities
  • =ban (tagged user) [reason] - Bans a user from the server
  • =esnipe [number] - Recover edited messages
  • =hackban (User ID) - Ban a user who is not currently in your server
  • =kick (tagged user) [reason] - Kick a user from the server
  • =mute (tagged user) [time] - Gives the tagged user the role 'muted'
  • =purge (number from 1 - 100) - Deletes messages
  • =snipe [number] - Recover deleted messages
  • =unban (Banned User ID) [reason] - Unbans a banned user from your server
  • =unbanall [reason] - Unbans every banned user from the server
  • =unmute (tagged user) - Removes the 'muted' role from the tagged user

Music

Commands which apply to voice channels
  • =bassboost [level] - Bassboost the song currrently playing!
  • =join/=j - Join the voice channel
  • =leave/=l - Leave the voice channel and stop the music
  • =loop/lp - Loop the current track!
  • =lyrics/=ly (song name) - Search for lyrics for a song
  • =nowplaying/=np - Check which song is currently playing
  • =pause/=p - pause the player
  • =play (song name) - Searches for your song and returns a selection to play from
  • =queue/=q - Check the queue and the next songs which will be played
  • =resume/=r - unpause the player
  • =shuffle/=sf - shuffle the queue
  • =skip/=s - Skip to the next track in the queue
  • =volume/=v (number) - Change the volume of the player (1-100)

Utility

Configuration commands for Frysta
  • =autobump - Toggle bump reminders
  • =autotranslate - toggle automatic translation
  • =create (time) (channel mention) (prize) - Create a giveaway
  • =custom-remove [command name] - Remove a custom command
  • =custom [command name] [command response] - Create a custom command!
  • =custominfo [command name or tagged user] - Get information about a custom command
  • =customs - View all custom commands in your server
  • =reminder (time) (reason) - Set a reminder
  • =t (language code) (language code) (text) - Translate
  • =vote (topic) - Set up a vote

Commands

Frysta has a lot of commands and some of them may be a little confusing so here's a list of all of the commands with a little description so you unserstand it.

Moderation

Frysta can help you manage your server

=ban - ban a tagged user from the server
=cwarns - clear the warns of a tagged user
=kick - kick a tagged user from the server
=mute - Mute a tagged user temporarily
=purge - delete messages (must specify the amount, 1-100)
=unban - unbans every banned user from the server
=warn - warn a tagged user for an offence
=warns - check how many times a tagged user has been warned in the server

Economy

It's like a game

=bal/=balance - check how much money a user has
=beg - beg for coins
=bet - gamble some coins
=daily - redeem a daily reward of 500 coins
=steal - steal coins from a user
=work - work and earn some money

Info

Find out about stuff

=credits - see who made the bot
=define - look up words in the dictionary
=info - find out about your server/a user
=recommend - unbiased song recommendations
=spy - find out all the information about a user

Generally useful

Commands which could be useful

=autobump - toggles autobump!
=absetup - Setup autobumper
=blockdm - toggles whether or not people can dm you with the =dm/=gdm command
=colour - view a colour of your choice (you need to provide a hex code)
=dm - use the bot to dm a user in your server (provide an ID)
=find - send a request for a user in your server
=help - this has some useful links
=raffle - use this to randomly raffle from a number
=t - translator, you must provide a language code to translate from and into. If you want to auto-detect the language use the language code 'dk'
=settings - view the settings for your server
=vote - hold a vote

Frozen

Frozen commands for frozen fans

=anna - see image manipulation
=elsa - see image manipulation
=elsanna - view posts from elsanna subreddit
=frozen - pictures from frozen
=intotheunknown - lyrics for into the unknown

=eime - Use webhooks to mimic a user but send an embed

=howgay - check how gay you are
=hug - hug a tagged user
=image - search for images
=joke - Frysta will tell a joke
=meme - view memes from reddit

=punish = punish Frysta for misbehaving
=punishc - see how many times you have punished Frysta for misbehaving
=roast - roast a tagged user
=say - get Frysta to say something

Image manipultion

To make images

Cubing

For people who like to solve Rubik's cubes

=2 - generate a 2 x 2 scramble
=3 - generate a 3 x 3 scramble
=time - Time a cube solve with our timer! - Press the spacebar to stop

Everything Else

The rest of the commands (yes there are more)

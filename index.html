FRYSTA

A Frozen Discord Bot




Translator

Frysta can translate to 20+ languages. There is automatic language detection and automatic translation to English

The format for translation is: =t [original language] [target language] [message]
For example: =t en de Hello, how are you? -> Hallo, wie geht's

The format for detecting the language is: =t dk [message]
For example: =t dk Estoy muy cansado -> I am very tired

Autotranslator works in a similar manner, and is toggled by =at

Webhooks

Frysta can use webhooks to mimic other users

The format for using webhooks is: =mime [target user] [message]
For example: =mime @Froz_n.anna Wow this bot is really cool

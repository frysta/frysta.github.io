Translator

We are able to translate from and into 20+ different languages for you. There is auto-detect and also an autotranslator!

To translate from one language to another you need their language codes. (This is the command: '=t (language code) (language code) (message you need to translate) - Note that the first language code it the one you translate from, the second is the one you translate to.)E.g. en (English) and de (German). If i type '=t en de Hello, how are you?' then the output will be 'Hallo, Wie geht's'. The autodetect will translate into English. To use it type '=t dk (message)' E.g. '=t dk etoy muy cansado' and the output will be 'I am very tired'. The autotranslator is similar and will translate every message into english. It can be toggled with the command '=at'.

Webhooks

You can use the comand '=mime' and the bot will create a webhook to mimic the user you tagged

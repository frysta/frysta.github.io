What

What is Frysta and why does it exist?

Frysta is a frozen discord bot. It is a multipurpose discord bot and has a wide range of features (check the commands page!) Frysta can be useful because it gives you accessebility to perform actions which otherwise would be very difficult with discord itself (for example unbanning every member can be done with Frysta in one command whereas with Discord you would have to go through the banned list and individually revoke each one. Another useful feature would be the translator - There is a feature which auotmatically translates messages in other languages into english, This is good because you do not have to open google translate (or DeepL) and type in the message; you can do it all from the comforts of discord.

Privacy Policy

What data we keep/don't

You may have noticed that some commands on Frysta show information about you - such as when your account was created. I can assure you that we will not sell any of your data to the Chinese government (The only data we keep are server / channel / user IDs). By using Frysta you are agreeing to let Frysta save your user ID (This is for using currency commands) and if you turn on server features such as autotranslate / Welcome messages your server ID and channel ID will be saved as well. If you want to wipe yourself from our database you can contact a developer through our support server.

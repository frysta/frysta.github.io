What

What is Frysta and why does it exist?

Frysta is a discord bot, based around Frozen 2 and the frozen franchise. It is a 'general multipurpose' bot which means that it does a bunch of epic stuff. It has moderation, currency, helpful search functions - basically everything you need to make your server epic. But why does Frysta exist? Frysta was created one day when i was assigned a project in English class (yes english class and not computing). The project was to expand a field of study and so i randomly chose to make a discord bot in js. In retrospect this probably was not the smartest idea since i did not know any javascript and was relatively new to discord.
